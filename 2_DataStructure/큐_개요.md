# 큐

## 개요

자료구조 시간에 배운 큐 맞음

### 사용법

파이썬
```python
from collections import deque

dq = deque()
dq.append(1)
dq.append(2)
dq.append(3)
print(dq.popleft())
print(dq.popleft())
print(dq.popleft())
```
